Quer abrir uma loja e não tem produtos para vender? Sem problemas. O plugin Loja Automática cria automaticamente uma loja com produtos do Mercado Livre, onde você recebe comissão por vendê-los.

== Description ==

**Principais Recursos:**

1. Crie sua loja facilmente.
2. Defina que tipos de ofertas serão vendidos na loja.
3. Defina o tempo entre as importações de ofertas.
4. Defina entre o WP Cron para importação ou o Unix Cron.
5. Utilize um dos temas prontos no site oficial do plugin na seção [Temas para Loja Automática](http://www.lojaautomatica.com.br/temas) ou crie o seu.

**Mais Informações:**

* Acesse o [Site do Plugin Loja Automática](http://www.lojaautomatica.com.br/) para maiores informações e demonstrações

== Installation ==

1. Envie os conteúdos do arquivo loja-automatica.zip para a sua pasta de plugins.
2. Ative o plugin.
3. Um menu "Loja Automática" vai aparecer no seu painel. 
4. Defina as categorias de ofertas de seu blog em "Criar Categorias".
3. Edite seu tema ou utilize um dos temas prontos no site oficial do plugin na seção [Temas para Loja Automática](http://www.lojaautomatica.com.br/temas)
5. E isso é tudo. Você terá uma loja automática e muito rentável!

Para uma explicação mais detalhada, acesse o [Guia de Instalação e Configuração do Loja Automática](http://www.lojaautomatica.com.br/instalacao-e-configuracao).
