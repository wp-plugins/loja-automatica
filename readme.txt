Quer abrir uma loja e não tem produtos para vender? Sem problemas. O plugin Loja Automática cria uma loja automática para você.

== Description ==

Você tem um bom domínio e que ganhar uma grana com ele? O melhor a fazer é abrir uma loja.

Não tem produtos? Não tem tempo? Não tem capital?

Sem problemas. O plugin Loja Automática transforma uma instalação do Wordpress em uma loja que vende produtos do Mercado Livre onde você ganha comissão pelas vendas. O único trabalho que você vai ter é de definir quais os tipos de produtos que serão vendidos na loja.

Se você duvida que pode dar certo, sugiro dar uma olhada no que escrevi em meu blog sobre ter [ganhado mais de R$ 9.000 em um ano com o Mercado Sócios](http://blog.gesing.com.br/2012/04/mercadosocios-recebi-r-9-12927-em-um-ano/).

**Principais Recursos:**

1. Crie sua loja facilmente.
2. Defina que tipos de ofertas serão vendidos na loja.
3. Defina o tempo entre as importações de ofertas.
4. Defina entre o WP Cron para importação ou o Unix Cron.
5. Utilize um dos temas prontos no site oficial do plugin na seção [Temas para Loja Automática](http://www.lojaautomatica.com.br/temas) ou crie o seu.

**Mais Informações:**

* Acesse o [Site do Plugin Loja Automática](http://www.lojaautomatica.com.br/) para maiores informações e demonstrações

== Installation ==

1. Envie os conteúdos do arquivo loja-automatica.zip para a sua pasta de plugins.
2. Ative o plugin.
3. Um menu "Loja Automática" vai aparecer no seu painel. 
4. Defina as categorias de ofertas de seu blog em "Criar Categorias".
3. Edite seu tema ou utilize um dos temas prontos no site oficial do plugin na seção [Temas para Loja Automática](http://www.lojaautomatica.com.br/temas)
5. E isso é tudo. Você terá uma loja automática e muito rentável!

Se você estiver com dificuldades de adaptar seu plugin, utilize um dos temas prontos disponíveis no site oficial do plugin na seção [Temas para Loja Automática](http://www.lojaautomatica.com.br/temas)

Para uma explicação mais detalhada, acesse o [Guia de Instalação e Configuração do Loja Automática](http://www.lojaautomatica.com.br/instalacao-e-configuracao).
